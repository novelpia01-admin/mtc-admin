Section 01 수강 신청 방법
🖥️STEP 01
홈페이지 접속 후 회원가입 필수

아래 주소로 접속한 뒤 오른쪽 상단 [회원가입] 버튼을 클릭하세요.
회원가입 시 핸드폰 번호를 정확히 입력해야 합니다. 수업 안내 문자 발송에 사용됩니다.

🔗 mtc.easytalk.co.kr  ↗
👆STEP 02
수강신청

로그인 후 메인 화면 오른쪽 상단의 [수강신청] 버튼을 클릭하세요.

📚STEP 03
언어 · 방식 · 과목 선택
  • 언어 — 영어, 일본어 등 원하는 언어를 선택하세요.
  • 수업 방식 — 전화 또는 화상을 선택하세요.
  • 과목 대분류 — 정규회화 / 비즈니스 / 시험준비 / 뉴스미디어 / 자유주제 중 선택하세요.
처음 신청이라면? 신규 학습자에 한해 첫 수업은 레벨 테스트로 진행됩니다.
수업 내용을 변경하고 싶다면? 고객센터 1644-0507로 문의하시면 변경 가능합니다.
⏱️STEP 04
수업 시간 · 횟수 선택
🎯
Option A
주 1회 · 20분
한 번에 집중해서 알차게 수업
🔁
Option B
주 2회 · 10분
짧고 자주, 꾸준한 습관 형성
📆STEP 05
수업 요일 · 희망 시간 선택
  • 주 1회 (20분) — 원하는 요일 및 희망 시간을 직접 선택할 수 있어요.
  • 주 2회 (10분) — 별도 선택이 없으면 화요일 + 목요일로 자동 배정됩니다.
공휴일은 수업이 없습니다. 해당 수업은 보강으로 진행됩니다.
✅STEP 06
신청 완료 & 확인 마지막 단계
  • Step 01~03 선택 완료 후 [신청하기] 버튼을 클릭하세요.
  • 신청 직후 카카오 알림톡 (이지톡)으로 확인 문자가 발송됩니다.
  • 강사 배정 후 별도 안내 문자가 추가로 발송됩니다.
연기·취소는 수업 시작 1시간 전까지만 가능합니다. 이후엔 결석 처리될 수 있으니 시간을 꼭 확인해 주세요.
